# AI Career Mentor 🚀

An AI-powered Career Mentor that helps students and professionals improve their resumes, prepare for interviews, analyze skills, and generate personalized career roadmaps.

## Features

- 🤖 AI Resume Review
- 📄 ATS Score Analysis
- 🎯 Career Roadmap Generator
- 💼 Mock Interview
- 📚 Course Recommendations
- 📊 Job Readiness Dashboard
- 🔒 Secure Authentication
- ☁️ Cloud Deployment
